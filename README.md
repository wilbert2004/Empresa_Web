Empresa Web - Beauty salon 
Este proyecto es una página web de maquillaje desarrollada como práctica de trabajo con ramas en GitHub.

Secciones del sitio

- Home: Página principal
- Nosotros: Información sobre la empresa
- Servicios: Tipos de servicios de maquillaje
- Contacto: Formulario o datos de contacto

 Objetivo
Aplicar el uso de ramas en Git para el desarrollo colaborativo de un sitio web.
